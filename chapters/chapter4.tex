\section{Experimental Setup}
This chapter presents the experimental results for the four core model configurations defined in Chapter 3. The goal is to compare architecture choice and training strategy under a common evaluation setting. All experiments use the same binary classification task, the same preprocessing pipeline, and the same train-validation-test split. This ensures that any observed performance differences can be interpreted mainly in terms of the model architecture or the use of transfer learning rather than changes in data preparation.

The four experiments evaluated in this thesis are listed below:
\begin{itemize}
    \item E1: ResNet50 trained from scratch;
    \item E2: ResNet50 with transfer learning;
    \item E3: DenseNet121 trained from scratch;
    \item E4: DenseNet121 with transfer learning.
\end{itemize}

The dataset split, preprocessing parameters, batch size, optimizer, learning rate, training epochs, and checkpoint rule follow the methodology described in the previous chapter.
