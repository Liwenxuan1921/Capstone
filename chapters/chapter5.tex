\section{Implications for Medical Image Classification}
Explain what the results imply for practical chest X-ray classification research. For example, if transfer learning produces clear improvements, discuss why pretrained visual representations may be helpful even when the source domain differs from medical imaging. If Grad-CAM shows reasonable attention patterns, explain how that supports model transparency while also noting its limitations.

\section{Limitations}
This thesis has several expected limitations that should be acknowledged explicitly. These include label noise in NIH ChestXray14, the simplification from multi-label diagnosis to binary classification, the limited number of model architectures tested, and the qualitative nature of Grad-CAM interpretation. The discussion should state how these limitations affect the generalizability of the conclusions.
